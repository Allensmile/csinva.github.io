Backpropagation algorithm

The backpropagation algorithm is essential for training large neural networks quickly. This article explains how the algorithm works.

Simple neural network

On the right, you see a neural network with one input, one output node and two hidden layers of two nodes each.

Nodes in neighboring layers are connected with weights [Math Processing Error]wij, which are the network parameters.

Activation function

Each node has a total input [Math Processing Error]x, an activation function [Math Processing Error]f(x) and an output [Math Processing Error]y=f(x). [Math Processing Error]f(x) has to be a non-linear function, otherwise the neural network will only be able to learn linear models.

A commonly used activation function is the Sigmoid function: [Math Processing Error]f(x)=11+e−x.

Error function

The goal is to learn the weights of the network automatically from data such that the predicted output [Math Processing Error]youtput is close to the target [Math Processing Error]ytarget for all inputs [Math Processing Error]xinput.

To measure how far we are from the goal, we use an error function [Math Processing Error]E. A commonly used error functon is [Math Processing Error]E(youtput,ytarget)=12(youtput−ytarget)2.

Forward propagation

We begin by taking an input example [Math Processing Error](xinput,ytarget) and updating the input layer of the network.

For consistency, we consider the input to be like any other node but without an activation function so its output is equal to its input, i.e. [Math Processing Error]y1=xinput.

Forward propagation

Now, we update the first hidden layer. We take the output [Math Processing Error]y of the nodes in the previous layer and use the weights to compute the input [Math Processing Error]x of the nodes in the next layer.

Using these 2 formulas we propagate for the rest of the network and get the final output of the network.

Error derivative

The backpropagation algorithm decides how much to update each weight of the network after comparing the predicted output with the desired output for a particular example. For this, we need to compute how the error changes with respect to each weight [Math Processing Error]dEdwij.
Once we have the error derivatives, we can update the weights using a simple update rule:
[Math Processing Error]wij=wij−αdEdwij
where [Math Processing Error]α is a positive constant, referred to as the learning rate, which we need to fine-tune empirically.

[Note] The update rule is very simple: if the error goes down when the weight increases ([Math Processing Error]dEdwij<0), then increase the weight, otherwise if the error goes up when the weight increases ([Math Processing Error]dEdwij>0), then decrease the weight.

Additional derivatives

To help compute [Math Processing Error]dEdwij, we additionally store for each node two more derivatives: how the error changes with:
  • the total input of the node [Math Processing Error]dEdx and
  • the output of the node [Math Processing Error]dEdy.

Back propagation

Let's begin backpropagating the error derivatives. Since we have the predicted output of this particular input example, we can compute how the error changes with that output. Given our error function [Math Processing Error]E=12(youtput−ytarget)2 we have:
[Math Processing Error]∂E∂youtput=youtput−ytarget

Back propagation

Now that we have [Math Processing Error]dEdy we can get [Math Processing Error]dEdx using the chain rule.
[Math Processing Error]∂E∂x=dydx∂E∂y=ddxf(x)∂E∂y
where [Math Processing Error]ddxf(x)=f(x)(1−f(x)) when [Math Processing Error]f(x) is the Sigmoid activation function.

Back propagation

As soon as we have the error derivative with respect to the total input of a node, we can get the error derivative with respect to the weights coming into that node.
[Math Processing Error]∂E∂wij=∂xj∂wij∂E∂xj=yi∂E∂xj

Back propagation

And using the chain rule, we can also get [Math Processing Error]dEdy from the previous layer. We have made a full circle.
[Math Processing Error]∂E∂yi=∑j∈out(i)∂xj∂yi∂E∂xj=∑j∈out(i)wij∂E∂xj

Back propagation

All that is left to do is repeat the previous three formulas until we have computed all the error derivatives.
